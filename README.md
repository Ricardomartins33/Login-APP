 # 🔐 Login APP

Projeto de interface de login moderna, responsiva e funcional, desenvolvido com foco em boas práticas de Front-End.

## 📌 Sobre o projeto

O **Login APP** é um sistema de autenticação visual, criado para demonstrar habilidades na construção de interfaces limpas, organizadas e adaptáveis a diferentes dispositivos.

O objetivo do projeto é simular uma tela de login real, com design profissional e experiência de usuário agradável.

---

## 🚀 Tecnologias utilizadas

- HTML 
- CSS3  
- Design Responsivo  
- Git e GitHub  

---

## 💻 Funcionalidades

✔️ Interface de login moderna e intuitiva  
✔️ Layout responsivo (Desktop e Mobile)  
✔️ Estrutura organizada e de fácil manutenção  
✔️ Código limpo e reutilizável
